# vite-vue3-ts-template

## Project setup
```
npm install
```

### 本地环境运行
```
npm run serve
```

### 开发环境构建
```
npm run build:dev
```

### 测试环境构建
```
npm run build:test
```

### 生产环境构建
```
npm run build:prod
```



## 项目开发
### Icons
项目用到的icon图标使用 `.svg` 后缀保存到 **src/assets/icons** 目录下，使用时直接用自定义组件的方式：
```
 <svg-icon :name="add"></svg-icon> 
```
